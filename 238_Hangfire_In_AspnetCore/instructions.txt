2. Run the migration " dotnet ef database update 'initialize db' " & you will have your database
3. Nuget Packages are following
    a.Hangfire.AspNetCore
    b.Hangfire.SqlServer
    c.Microsoft.Data.SqlClient
    d.Microsoft.EntityFrameworkCore
    e.Microsoft.EntityFrameworkCore.SqlServer
4. Create Background Service and settle the code whatever you want
5. Check Program.cs
6. /Hangfire is the dashboard route of hangfire.
7. You have been provided with basic db.sql script so you can use the same databaes i used .
